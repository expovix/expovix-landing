# ExpoVix Landing — Claude Code Instructions

## Project
- Product: ExpoVix (expovix.com)
- Repo: expovix/expovix-landing
- Stack: React + Vite
- Deployment: Vercel (auto-deploy on merge to main)

## Workflow
1. Read TASKS.md for the current task
2. Execute the task autonomously
3. Create a PR using: gh pr create --title "task title" --body "what was done"
4. Never push directly to main

## Rules
- Mobile-first always
- All colors via CSS variables only
- Brand accent: #FF5F29 orange
- No navy, no teal, no green
- Auth routes: expovix.com/login and expovix.com/register (Supabase)
- App lives at: app.expovix.com

## PR Format
Title: [scope] brief description
Body: what changed, why, files affected
